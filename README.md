# CalmAI 🌿 — Anxiety Support Companion 

A calming, anxiety support web app with an AI chat experience, guided breathing exercises, and 5-4-3-2-1 grounding.

## Live Demo
- **Frontend (Vercel):** [(https://calm-ai-4sl9-git-main-ex0ternal190-4599s-projects.vercel.app/) ] 
- **Backend (Render):** [(https://calmai.onrender.com/)]

## Description
- CalmAI was created for the quiet moments when anxiety feels loud and overwhelming. It exists to offer a calm, steady presence when no one else may be available, a space where users can breathe, feel heard, and gently regain control. Through supportive conversation and simple grounding tools, CalmAI helps turn moments of panic into moments of pause. 

## Features
- 💬 **AI Chat**: supportive, structured responses (non-clinical, no diagnosis)
- 🫁 **Breathing Exercises**: Box breathing / 4-7-8 / Calm breath with animated timer
- 🌱 **Grounding (5-4-3-2-1)**: step-by-step sensory grounding flow
-  **Crisis keyword detection**: shows immediate helpline guidance when needed
-  **UI**: glassmorphism + animated orbs + smooth interactions

## Tech Stack
**Frontend:** React, Axios, CSS (custom UI)  
**Backend:** FastAPI, Uvicorn  
**AI:** Groq (OpenAI-compatible client)  
**Deployment:** Vercel (frontend) + Render (backend)
